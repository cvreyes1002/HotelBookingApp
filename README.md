Hotel Booking App
How to make a Complete Hotel Booking Website using React JS, Hotel Booking app in react js and tailwind css

https://www.youtube.com/watch?v=QSjHs5Qd-GQ&t=67s

Build a complete Hotel Booking website using React JS and Tailwind CSS. This is a Hotel booking web app where users can book hotels. Users can also signup as a hotel partner to list their hotel rooms and manage the hotel bookings from the admin panel.
Create the Login and registration feature using Clerk. Clerk is a user authentication and user management platform. It  also provides beautiful and ready to use UI components like Sign-In, Sign-up and profile component for React app.
